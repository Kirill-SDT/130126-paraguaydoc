# ParaguayDoc - Landing Page

Профессиональный лендинг для иммиграционной компании ParaguayDoc, специализирующейся на оформлении документов для переезда в Парагвай для русскоязычных клиентов.

## Описание проекта

ParaguayDoc - это современный, конверсионно-оптимизированный одностраничный сайт (landing page) с полным функционалом для привлечения клиентов, интересующихся иммиграцией в Парагвай.

### Основные возможности

- ✅ Полностью адаптивный дизайн (мобильные, планшеты, десктопы)
- ✅ 10 функциональных секций с подробной информацией
- ✅ Контактная форма с валидацией
- ✅ FAQ с аккордеоном
- ✅ Плавная навигация и анимации
- ✅ Доступность (ARIA labels, семантический HTML)
- ✅ SEO-оптимизация (meta tags, семантическая разметка)
- ✅ Современный дизайн в цветах флага Парагвая

## Секции лендинга

1. **Header (Шапка)** - Логотип, навигационное меню, мобильное меню
2. **Hero (Главный экран)** - Заголовок, описание, CTA кнопки, ключевые преимущества
3. **Services (Услуги)** - 6 основных услуг с описанием и списком включенного
4. **Pricing (Тарифы)** - 3 тарифных плана с прозрачным ценообразованием
5. **Timeline (Этапы работы)** - 5 шагов процесса оформления документов
6. **Trust (Доверие)** - Почему выбирают нас, статистика компании
7. **FAQ (Вопросы-ответы)** - 8 популярных вопросов с интерактивным аккордеоном
8. **Contact (Контактная форма)** - Форма обратной связи с валидацией
9. **Additional Services (Доп. услуги)** - 6 дополнительных сервисов
10. **Footer (Подвал)** - Контактная информация, навигация, социальные сети

## Быстрый старт

### Локальное использование

1. Распакуйте файлы в любую папку
2. Откройте `index.html` в браузере
3. Готово! Сайт работает без сервера

### Использование с локальным сервером

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (установите http-server: npm install -g http-server)
http-server -p 8000

# PHP
php -S localhost:8000
```

Затем откройте браузер по адресу: `http://localhost:8000`

## Развертывание

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы в репозиторий
3. Перейдите в Settings → Pages
4. Выберите ветку (main/master) и папку (root)
5. Сохраните изменения
6. Ваш сайт будет доступен по адресу: `https://username.github.io/repository-name`

### Netlify

1. Перетащите папку с файлами на [Netlify Drop](https://app.netlify.com/drop)
2. Или подключите Git репозиторий
3. Сайт автоматически развернется с HTTPS

### Vercel

```bash
# Установите Vercel CLI
npm i -g vercel

# В папке с проектом
vercel

# Следуйте инструкциям
```

### Хостинг (традиционный)

1. Загрузите файлы на хостинг через FTP/SFTP
2. Поместите файлы в корневую директорию (public_html, www, htdocs)
3. Убедитесь, что index.html находится в корне

## Настройка

### Изменение контактной информации

Отредактируйте `index.html`:

```html
<!-- Найдите секцию contact и обновите -->
<p><a href="mailto:ваш@email.com">ваш@email.com</a></p>
<p><a href="tel:+595123456789">+595 123 456 789</a></p>
```

### Подключение формы к backend

В файле `script.js` найдите функцию `submitForm()` и замените симуляцию на реальный API запрос:

```javascript
fetch('/api/contact', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(data => {
    // Обработка успеха
})
.catch(error => {
    // Обработка ошибки
});
```

### Подключение Google Analytics

Добавьте перед закрывающим тегом `</head>` в `index.html`:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```

### Изменение цветов

Отредактируйте CSS переменные в `styles.css`:

```css
:root {
    --color-blue: #0038A8;      /* Основной цвет */
    --color-red: #D52B1E;        /* Акцентный цвет */
    /* ... остальные переменные */
}
```

## Оптимизация

### Производительность

Сайт уже оптимизирован:
- ✅ Минимальные HTTP запросы
- ✅ CSS и JS оптимизированы
- ✅ Нет тяжелых библиотек

- ✅ Плавные анимации с CSS

### Дополнительная оптимизация (опционально)

1. **Минификация CSS/JS**
   ```bash
   # Используйте онлайн инструменты или:
   npm install -g clean-css-cli uglify-js
   cleancss -o styles.min.css styles.css
   uglifyjs script.js -o script.min.js -c -m
   ```

2. **Добавление изображений**
   - Используйте WebP формат для лучшего сжатия
   - Оптимизируйте изображения перед загрузкой
   - Используйте CDN для статических ресурсов

3. **Кэширование**
   Добавьте `.htaccess` для Apache:
   ```apache
   <IfModule mod_expires.c>
     ExpiresActive On
     ExpiresByType text/css "access plus 1 year"
     ExpiresByType application/javascript "access plus 1 year"
   </IfModule>
   ```
